## Task 4: k-trials / multi-step mapping

**Default mapping (all packs including this):** k-trials implemented as OUTER LOOP.
- A Harbor `task` (this dir) = **one trial**.
- To get k trials: driver (e.g. `scripts/harbor_run_job.py`) runs the task k times (different seeds/personas via mock or env).
- Dataset in Harbor sense can be k copies if using Harbor dataset feature; equivalent to separate runs labeled by job+ trial.
